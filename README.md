# 🐾 Rebbit Hole

**The official game launcher for Reb's studio — a unified hub for all current and future game titles.**

Rebbit Hole is a desktop launcher that handles authentication, game installation, patching, cross-game rewards, and player identity across all games in the studio ecosystem.

---

## What is this?

Rebbit Hole is the studio's own launcher platform, designed to work alongside (or independently of) platforms like Steam. Players create one account and carry their identity, rewards, and achievements across every game released under the studio.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Launcher Client | React + Vite + TypeScript |
| Desktop Wrapper | Tauri (Rust) |
| Backend API | FastAPI (Python) |
| Auth & Identity | Supabase |
| File Hosting / CDN | Cloudflare R2 |
| Source Control | GitHub |

## Core Features

- **Authentication** — Sign-up, login, and session management via Supabase
- **Game Library** — Browse, install, uninstall, and launch games
- **Patch System** — Version checking and incremental file updates
- **Reward System** — Cross-game achievements, titles, and cosmetics tied to a global player ID
- **Community (planned)** — News feed, announcements, player profiles

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+
- [Rust](https://www.rust-lang.org/tools/install) (required by Tauri)
- [Python](https://www.python.org/) 3.11+
- A [Supabase](https://supabase.com/) project

### Launcher (Development)

```bash
cd launcher
npm install
npx tauri dev
```

### Backend (Development)

```bash
cd backend
python -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```
